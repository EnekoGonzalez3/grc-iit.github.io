### Install Spack
```bash
cd ${HOME}
git clone https://github.com/spack/spack.git
cd spack
git checkout tags/v0.22.2
echo ". ${PWD}/share/spack/setup-env.sh" >> ~/.bashrc
source ~/.bashrc
```

### Add IOWarp to Spack
```bash
spack repo add iowarp-install/iowarp-spack
```

## Install IOWarp

```bash
spack install iowarp
```
